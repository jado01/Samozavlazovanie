# Obrázky pre dokumentáciu

Tento priečinok je určený iba pre fotografie, schémy a grafiku vhodnú na zverejnenie v GitHub repozitári.

Pred pridaním fotografie:

1. skontrolovať, či nezobrazuje osoby, mená, adresu alebo citlivé časti domácnosti,
2. odstrániť EXIF vrátane GPS,
3. orezať nepotrebné okolie,
4. primerane zmenšiť rozlíšenie a veľkosť súboru,
5. použiť opisný názov súboru.
