# Originálne fotografie – iba lokálne

Sem skopíruj pôvodné fotografie z telefónu alebo fotoaparátu. Obsah tohto priečinka je okrem tohto návodu ignorovaný Gitom, pretože originály môžu obsahovať EXIF/GPS údaje alebo súkromné časti domácnosti.

Odporúčaný názov:

`YYYY-MM-DD_poradove-cislo_popis.jpg`

Príklad:

`2026-07-03_01_balkon-pohlad-od-dveri.jpg`

Vybrané fotografie určené na zverejnenie sa po kontrole a odstránení citlivých údajov skopírujú do `docs/images/`.
